 WORKTREE WORKFLOW
 ============================================================================

 main
  └── feature/my-feature  (PR branch — integration point)
        ├── feature/my-feature/auth-logic      (worktree 1)
        ├── feature/my-feature/api-endpoints   (worktree 2)
        └── feature/my-feature/ui-components   (worktree 3)

 Worktrees live in .worktrees/ at the repo root (gitignored).
 Shell helpers: gwa (add), gwl (list/switch), gwr (remove), gwr -D (remove+delete branch)

 1. Create feature branch + worktrees

    git checkout -b feature/my-feature origin/main
    gwa feature/my-feature/auth-logic
    gwa feature/my-feature/api-endpoints
    gwa feature/my-feature/ui-components

    Or use ` + G to create worktrees interactively from tmux.


 2. Run Claude Code in parallel (one tmux window per worktree)

    # ` + g to fzf-switch between worktrees, then run claude in each
    claude   # "Implement the authentication service for contract sharing"
    claude   # "Implement the REST API endpoints for contract sharing"
    claude   # "Implement the frontend components for contract sharing"


 3. Merge sub-branches into feature branch (from any worktree)

    git checkout feature/my-feature

    git merge feature/my-feature/auth-logic

    # Rebase later branches if they depend on earlier ones
    git rebase feature/my-feature feature/my-feature/api-endpoints
    git merge feature/my-feature/api-endpoints

    git rebase feature/my-feature feature/my-feature/ui-components
    git merge feature/my-feature/ui-components


 4. Clean up and submit PR

    gwr -D feature/my-feature/auth-logic
    gwr -D feature/my-feature/api-endpoints
    gwr -D feature/my-feature/ui-components

    git push -u origin feature/my-feature

 Tip: git worktree prune — cleans stale refs if a worktree dir was deleted manually


 TMUX CHEAT SHEET                                    prefix = ` (backtick)
 ============================================================================

 BASICS
   ` + d           Detach from session
   ` + ?           Show this cheat sheet (popup)
   ` + r           Reload tmux config
   ` + e           Send literal backtick

 SESSIONS
   ` + f           Sessionizer (fzf project picker → create/switch session)
   ` + s           List sessions
   ` + $           Rename session
   tmux new -s X   Create session named X
   tmux a -t X     Attach to session X

 WINDOWS
   ` + c           New window (keeps current path)
   ` + 1-9         Jump to window N
   ` + ,           Rename window
   ` + w           Window chooser
   ` + &           Kill window

 PANES
   ` + |           Split vertical (keeps path)
   ` + -           Split horizontal (keeps path)
   ` + h/j/k/l    Navigate panes (vim-style)
   ` + H/J/K/L    Resize pane (repeatable)
   ` + z           Toggle pane zoom
   ` + x           Kill pane
   ` + q           Show pane numbers
   ` + !           Move pane to its own window

 COPY MODE  (` + [  to enter, vi keys)
   v               Begin selection
   y               Copy to clipboard
   q / Escape      Exit copy mode

 GIT WORKTREES
   ` + g           List/switch worktrees (fzf popup)
   ` + G           Create worktree → new tmux window

 WINDOW NAMING
   Windows auto-rename to dir:branch (or dir basename if not git).
   Detached HEAD shows dir:SHA. Updates every 5 min.
   ` + ,           Manual rename (disables auto-rename for that window)

 SESSION PERSISTENCE (resurrect + continuum)
   ` + Ctrl-s         Save session (manual)
   ` + Ctrl-r         Restore session (manual)
   Auto-saves every 15 min. Auto-restores on tmux server start.

 SHELL: gwa <branch>  gwl  gwr [-D] <branch>
 MOUSE: enabled (scroll, click, resize)
